# Landuse_Classification_GEE

基于Google Earth Engine进行土地利用分类，演示代码为基于Landsat7的随机森林分类代码，具体操作请参考handbook，代码主体为js文件/n

Google Earth Engineによる土地利用分類、デモコードはLandsat7に基づくランダムフォレスト分類コードで、具体的な操作はハンドブックを参照してください。
